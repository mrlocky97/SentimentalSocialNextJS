# 🔐 INSTRUCCIONES PARA CONFIGURAR CREDENCIALES REALES DE TWITTER

## 📝 PASO 1: Edita el archivo .env.local

Abre el archivo `.env.local` en la raíz del proyecto y REEMPLAZA estas líneas:

```bash
# ANTES (valores de ejemplo):
TWITTER_USERNAME=tu_usuario_real_de_twitter_aqui
TWITTER_PASSWORD=tu_contraseña_real_de_twitter_aqui  
TWITTER_EMAIL=tu_email_real_de_twitter@ejemplo.com

# DESPUÉS (TUS DATOS REALES):
TWITTER_USERNAME=miusuario123
TWITTER_PASSWORD=micontraseñasupersecreta
TWITTER_EMAIL=miemail@gmail.com
```

## ⚠️ IMPORTANTE:
- NO uses @ en el username (solo: miusuario123, no @miusuario123)
- USA tu contraseña real de Twitter
- USA el email asociado a tu cuenta de Twitter

## 🔍 CÓMO VERIFICAR QUE FUNCIONA:

Después de configurar las credenciales, ejecuta:
```bash
npm run test:scraping
```

Si funciona verás:
```
✅ Successfully authenticated with Twitter!
✅ Successfully scraped from Twitter
```

Si no funciona verás:
```
⚠️ Twitter credentials not found in environment variables
⚠️ Using mock scraper
```

## 🛡️ SEGURIDAD:
- ✅ El archivo .env.local NO se sube a Git
- ✅ Tus credenciales están seguras
- ✅ Puedes usar una cuenta secundaria si prefieres

## 🚀 ¡LISTO!
Una vez configurado, tendrás acceso a datos REALES de Twitter con análisis de sentimientos en tiempo real.
